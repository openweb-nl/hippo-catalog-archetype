hippo-catalog-archetype
=======================

For Linux:

mvn archetype:generate \
-DarchetypeGroupId=nl.openweb.archetype \
-DarchetypeArtifactId=hippo-catalog-archetype \
-DarchetypeVersion=1.0-SNAPSHOT

For Windows:

mvn archetype:generate ^
-DarchetypeGroupId=nl.openweb.archetype ^
-DarchetypeArtifactId=hippo-catalog-archetype ^
-DarchetypeVersion=1.0-SNAPSHOT
